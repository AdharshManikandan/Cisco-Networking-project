# Cisco Networking Project — TechVille College Network Design

## 📖 Overview
This project presents the design and implementation of a complete computer network for **TechVille College**, built and simulated using **Cisco Packet Tracer**. The goal is to create a **secure, reliable, and scalable** network system that connects all departments across the institution, supporting academic, administrative, and communication needs.

## 🎯 Objectives
- Design a hierarchical and scalable network architecture for a multi-department college campus
- Ensure secure connectivity between administrative, academic, and communication systems
- Implement reliable routing and switching to minimize downtime and network congestion
- Provide a foundation that can be expanded as the institution grows

## 🏫 Network Scope
